⚙️ เชื่อมต่อ Notion

วิธีสร้าง API Key (ทำครั้งเดียว)

1. ไปที่ notion.so/my-integrations
2. กด "+ New integration" → ตั้งชื่อ "Port Dashboard" → Submit
3. Copy "Internal Integration Token" (ขึ้นต้นด้วย ntn_...)
4. ใน Notion เปิด database พอร์ตหุ้นอเมริกา → ⋯ → Connections → เชื่อม "Port Dashboard"
5. วาง token ด้านล่าง แล้วกด เชื่อมต่อ

Token จะเซฟในเครื่องเท่านั้น ไม่ถูกส่งออกไปไหน
