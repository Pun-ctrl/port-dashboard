⚙️ เชื่อม Notion API

ใส่ API Key จาก notion.so/my-integrations
ระบบจะโหลด R1–S4 ของทุกหุ้นจาก database พอร์ตหุ้นอเมริกา

วิธีสร้าง Integration:
1. ไปที่ notion.so/my-integrations → "New integration"
2. ตั้งชื่อ เช่น "Port Dashboard" → Create
3. คัดลอก Internal Integration Secret มาวางที่นี่
4. เปิด database พอร์ตหุ้นอเมริกา → Share → เพิ่ม integration
